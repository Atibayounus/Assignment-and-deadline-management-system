# Assignment and Deadline Management System

## Project Description
This project is a console-based Assignment and Deadline Management System designed to help students organize their assignments and track deadlines efficiently. It allows users to store assignment details using file handling and view their status easily.

## Objectives
- Manage assignments in an organized way  
- Track deadlines accurately  
- Reduce chances of missing submissions  
- Improve time management skills  

## Features
- Add new assignments  
- View all assignments  
- Track deadlines  
- Update assignment status (Pending / Completed)  
- Store data using text files  

## Technologies Used
- Programming Language: C / C++  
- File Handling: Text files  
- Interface: Console-based  

## assignment.txt Format
Assignment_ID | Subject | Assignment_Title | Deadline | Status  
101 | Physics | Hall Effect Assignment | 2025-01-10 | Pending  
102 | Programming | PF Project | 2025-01-15 | Completed  

## How to Run
1. Open the project in a C/C++ compiler  
2. Compile the source file  
3. Run the program  
4. Follow the menu options displayed
